ASL_Hand
The goal of this project is to allow LLMs to read sign language from users, then respond by signing through a robotic hand. Deaf people are often overlooked in many regards of communication, and chatbots are certainly an aspect that was not intended with accessibility as the primary function. This repo contains the software necessary to interpret sign language ysing 4 servos that will separately need to be integrated into a hand. The other portion of this project, which is the LLM understanding sign language, can be found in this forked repo: https://github.com/MiddleSender45/sign-language-interpreter.git.
Materials
- 4 pq12 linear actuators(or any pwm-servo)
- 1 arduino compatible microcontroller
- 1 PCA9685 pwm controller board
- 4 flex resistors
- 4 10k+ ohm resistors
- 2 push buttons
- external power supply (depending on servos)
- breadboard jumper cables
- 3d printer segments (optionally)
